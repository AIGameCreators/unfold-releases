## Overview
Release package for Unfold.

## Changes
- Improved reading guide behavior
- Refined block detail layout
- Stability improvements

## Installation
1. Download the `.vsix` file from Assets below
2. Open the Extensions view in your IDE
3. Select `Install from VSIX...`
4. Choose the downloaded file

## Notes
- Unfold is mainly tested on Visual Studio Code
- Behavior may differ in some VS Code fork IDEs
- Block splitting may be affected by differences in symbol support
